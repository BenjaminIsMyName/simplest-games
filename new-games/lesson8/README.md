## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. חיברו את הקבצים `style.css` ו-`game.js` בעזרת תגית `<link>` (ב-`<head>`) ו-`<script>` (בסוף ה-`<body>`).
3. עברו ל-`game.js` וחפשו קווים תחתונים (`___`) – מלאו אותם בהתאם לתגובות (חיבור ניקוד, עדכון תצוגות, חישוב אחוזים, הורדת חיים).
4. פתחו את `index.html` בדפדפן ולחצו על "הפעל סריקת קוד"; תתקנו את השגיאות עד שהמשחק יתחיל.
