## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. צרו שני שדות קלט (`<input>`): אחד עם `type="text"` ואחד עם `type="password"`.
3. הוסיפו לכל אחד `placeholder` ו-`class="cyber-input"`.
4. הוסיפו כפתור (`<button>`) שישלח את הטופס ויפעיל את הפריצה.
5. פתחו את הדף בדפדפן ולחצו על הכפתור; תקנו את השגיאות עד שהמשחק מתחיל.
