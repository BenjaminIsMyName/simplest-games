## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. החליפו את הטקסט בתוך תגית `<h1>` בטקסט חדש שלכם.
3. החליפו את הטקסט בתוך תגית `<h4>` (השם שלכם) בטקסט חדש.
4. פתחו את הדף בדפדפן וודאו שהכותרות החדשות מוצגות.
