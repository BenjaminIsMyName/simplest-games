## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. הוסיפו ל-`#hint-btn` את המאפיין `onclick="openVault()"` (משימה 1).
3. הוסיפו ל-`#secret-code` את המאפיין `oncopy="..."` (משימה 2).
4. פתחו את הדף בדפדפן ולחצו על הכפתור; תקנו את השגיאות עד שהכספת נפתחת והמשחק מתחיל.
