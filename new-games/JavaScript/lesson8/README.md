## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. חיברו את הקובץ `style.css` ל־HTML בעזרת תגית `<link>` בתוך `<head>`.
3. חיברו את הקובץ `game.js` ל־HTML בעזרת תגית `<script>` בסוף ה־`<body>`.
4. פתחו את `game.js` בעורך טקסט.
5. בשורה 2 (`const ___ = "חסר_קוד"`) – השאירו אותה כפי שהיא (מדובר בקוד בדיקה) ולא למחוק.
6. בשורה 19 (`score = ___ ;`) – השלימו את הביטוי כך שהניקוד יתעדכן (score + pointsEarned).
7. בשורה 23 (`document.getElementById("___").innerText = score;`) – הכניסו את ה‑ID המתאים מה‑HTML (שם המקום שבו מוצג הניקוד).
8. בשורה 29 (`let percentage = ___ ;`) – השלימו את הנוסחה: `totalCaptured / totalSeaBlocks * 100`.
9. בשורה 42 (`lives = ___ ;`) – השלימו את החיסור של 1 מהמשתנה `lives`.
10. פתחו את `index.html` בדפדפן ולחצו על "הפעל סריקת קוד"; תקנו את השגיאות עד שהמשחק יתחיל.
