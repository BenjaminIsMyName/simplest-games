## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. ודאו שהתמונה של החללית (`spaceship.png`) מחוברת נכון ב־`src` של תגית `<img>`.
3. ודאו שהתמונה של התרנגולת (`chicken.png`) מחוברת נכון ב־`src` של תגית `<img>`.
4. הוסיפו פסקה (`<p>`) עם טקסט קיים.
5. הוסיפו `<br>` לפני המילה "זהירות".
6. הוסיפו קישור פעיל (`<a href="..."></a>`) כדי ליצור קישור להתרעת תגבורת.
7. פתחו את הקובץ בדפדפן ולחצו על "🚀 התנע מערכות והתחל משחק"; תקנו את השגיאות עד שהמשחק יתחיל.
