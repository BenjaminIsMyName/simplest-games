## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. ודאו שלדף יש כותרת (`<title>`) ייחודית.
3. ודאו שיש קישור ל-favicon (`<link rel="icon">`).
4. שנה את צבע הטקסט של `#car-name` באמצעות CSS.
5. שנה את גודל הגופן של `#car-name` באמצעות CSS.
6. הוסף את התגית `<b>` בתוך הפסקת הכללים.
7. הוסף את התגית `<del>` בתוך הפסקת הכללים.
8. הוסף את התגית `<mark>` בתוך הפסקת הכללים.
9. עצב את `#my-car-body` כך שיהיה לו רוחב (width) וגובה (height).
10. עצב את `#my-car-body` כך שיהיה לו צבע רקע (background-color).
11. עצב את `#my-car-body` כך שיהיה לו גבול (border).
12. עצב את `#my-car-roof` כך שיהיה לו רוחב (width) וגובה (height).
13. עצב את `#my-car-roof` כך שיהיה לו צבע רקע (background-color).
14. עצב את `#my-car-roof` כך שיהיה לו גבול (border).
15. פתחו את הדף בדפדפן ולחצו על "🚦 הוצא רכב מהמוסך והתחל מרוץ"; תקנו את השגיאות עד שהמשחק יתחיל.
