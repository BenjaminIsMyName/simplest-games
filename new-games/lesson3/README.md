## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. ודאו שלדף יש כותרת (`<title>`) ייחודית ו-`<link rel="icon">` (favicon).
3. קשטו את שם הרכב (`#car-name`) באמצעות צבע וגודל גופן, והוסיפו בתוך הפסקת הכללים את התגיות `<b>`, `<del>` ו-`<mark>`.
4. בעיצוב (CSS) העניקו ל-`#my-car-body` ול-`#my-car-roof` width/height/background-color/border.
5. פתחו את הדף בדפדפן ולחצו על "🚦 הוצא רכב מהמוסך והתחל מרוץ"; תקנו את השגיאות עד שהמשחק יתחיל.
