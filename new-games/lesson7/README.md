## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. קבעו `money = 0` בתחילת הקובץ (משימה 1).
3. ב-`addIngredient`, הוסיפו 1 ל-`pitaFalafel` בכל לחיצה (משימה 2).
4. ב-`servePita`, הוסיפו את המשתנה `payment` ל-`money` (משימה 3).
5. ב-`buyTahini`, הורידו 100 מ-`money` (משימה 4a).
6. ב-`buyTahini`, שנו את `hasTahini` ל-`true` (משימה 4b).
7. פתחו את הדף בדפדפן ולחצו על "🔍 פתח דוכן"; תקנו את השגיאות עד שהמשחק מתחיל.
