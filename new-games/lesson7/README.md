## תרגיל (מה לעשות ב-index.html)

1. פתחו את `index.html` בעורך טקסט.
2. השלימו את קטעי הקוד שמסומנים ב-`___` בהתאם להערות:
   - קבעו `money = 0` בתחילת הקובץ.
   - ב-`addIngredient` הוסיפו 1 ל-`pitaFalafel`.
   - ב-`servePita` הוסיפו את `payment` ל-`money`.
   - ב-`buyTahini` הורידו 100 מ-`money` וגרמו ל-`hasTahini = true`.
3. פתחו את הדף בדפדפן ולחצו על "🔍 פתח דוכן"; תקנו את השגיאות עד שהמשחק מתחיל.
