# MPG GameZone

אתר משחקים חינוכיים של פרוגיקס (MPG)

## מה קיים
- דף בית: `index.html`
- סגנון: `styles.css`

## משחקים שמקושרים כרגע
- מילים באנגלית: `../games/vocab-memory/index.html`
- הקלדה עיוורת: `../games/typing/index.html`

## איך להוסיף משחק חדש
1. הוסיפו תת-תיקייה `mpg gamezone/games/<play-name>`
2. צרו שם קבצי `index.html` ו-`styles.css`/`index.js`
3. הוסיפו כרטיס חדש ב-`index.html` עם קישור ל-game
